###################
TECNOLOGIA APLICADA NO APRENDIZADO DA ANATOMIA HUMANA
###################

-------------------
Projeto de TCC referente ao curso de Sistemas de Informação, da instituição de ensino FHO - Araras/SP.

-------------------
O projeto consiste-se em um software de ensino-aprendizagem sobre a disciplina de Anatomia Humana. Onde o responsável poderá inserir categorias, subcategorias, fotos e descrições, sobre as variadas estruturas anatômicas. E o aluno, poderá acessar o sistema para visualizar as fotos inseridas, junto com suas descrições, a fim de auxiliá-lo no processo de aprendizagem em conjunto com o conteúdo transmitido pelo professor em sala de aula.
